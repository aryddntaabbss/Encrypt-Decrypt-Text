## Aplikasi Web Encrypt & Decrypt Text

Aplikasi web sederhana yang memungkinkan pengguna untuk **mengenkripsi** dan **mendekripsi** teks menggunakan metode **Caesar Cipher**. Aplikasi ini dibangun menggunakan **HTML, CSS, dan JavaScript**, serta bertujuan untuk menunjukkan penerapan algoritma enkripsi klasik dalam pengembangan web dasar.

### Fitur:
- **Enkripsi Teks**: Menggeser karakter sesuai nilai shift yang diberikan.
- **Dekripsi Teks**: Mengembalikan teks ke bentuk asli dengan menggeser karakter ke arah berlawanan.
- **Nilai Shift yang Dapat Diatur**: Pengguna bisa menentukan nilai shift antara 1 hingga 25.

### Cara Penggunaan:
1. Clone atau unduh repository ini.
2. Buka file `index.html` di browser Anda.
3. Masukkan teks, atur nilai shift, lalu klik **Encrypt** atau **Decrypt** untuk melihat hasilnya.

### Struktur File:
- `index.html`: Berisi struktur HTML aplikasi.
- `styles.css`: Mengatur tampilan antarmuka aplikasi.
- `script.js`: Berisi logika enkripsi dan dekripsi menggunakan algoritma Caesar Cipher.
